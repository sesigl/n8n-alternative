# Decision Records (DRs)

This directory contains Decision Records - formal proposals and documentation for significant decisions made in this project.

## What are Decision Records?

Decision Records capture important architectural, technical, and process decisions to maintain transparency, provide context for contributors, and track the evolution of the project.

## How to Use

For complete information about:
- When to write a DR
- How to propose a DR
- The DR lifecycle and process
- Guidelines and best practices

See [DR-001: Decision Record Process for Transparent Governance](./DR-001-decision-record-process.md).

Use [`decision-record-template.md`](./decision-record-template.md) as the starting point for all new DRs.
